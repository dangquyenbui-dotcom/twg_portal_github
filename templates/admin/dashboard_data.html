Dashboard Data Management

Download historical data from SQL Server and save as compressed files for instant dashboard loading and Excel exports. Downloaded data is stored on disk and can be copied to any server. The current year always loads live from SQL — only past years can be downloaded.

Bookings data is fetched from soytrn (historical orders). Each download pulls all raw line items (26 columns), aggregates them, and saves to bookings_dashboard_data/. After download, the dashboard and Excel exports read directly from disk — SQL Server is never queried again for that year.
