Dashboard Data Management

Download historical bookings data from SQL Server and save as compressed files for instant dashboard loading and Excel exports. Downloaded data is stored in the dashboard_data/ folder and can be copied to any server. The current year always loads live from SQL — only past years can be downloaded.

How it works: When you "Download" a year, the app fetches ALL raw line-item rows (26 columns) from soytrn in a single SQL query, processes and aggregates them on the hosting server, then saves everything — raw rows + pre-aggregated summary — as a gzip JSON file. After download, the dashboard loads the summary instantly (<1ms) and Excel exports read the raw rows directly from disk. SQL Server is never queried again for that year.
